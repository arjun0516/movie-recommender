# 🎬 Movie Recommendation System

A machine learning-based movie recommendation system that suggests similar movies based on user input.

## 🚀 Features
- Content-based recommendation system
- Uses TF-IDF vectorization and cosine similarity
- Interactive UI built with Streamlit

## 🧠 Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK
- Streamlit

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py

## 📌 Description
The system analyzes movie metadata such as overview, genres, keywords, cast, and crew to recommend similar movies.

## 📷 Output
Select a movie from the dropdown and get top 5 similar movie recommendations instantly.
